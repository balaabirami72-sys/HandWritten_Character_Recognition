# 📝 Intelligent Handwritten Word Recognition System

An interactive deep learning and computer vision web application that processes, segments, and auto-corrects handwritten text from real-world images.

🚀 **Live App Link:** 
https://handwrittencharacterrecognition-123.streamlit.app/

## 🛠️ Tech Stack
- **Frontend & Deployment:** Streamlit / Streamlit Community Cloud
- **Deep Learning Core:** TensorFlow / Keras (CNN Alphanumeric Classification)
- **Image Processing:** OpenCV-Python / Pillow / NumPy
